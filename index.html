What is Git?

Creator: Linus Torvalds

What is Git?

  • Provides information on:
    • Who changed what
    • When they changed it
    • Why they changed it
      (hopefully a valuable commit message)

What is Git?

  • Allows people to collaborate on a set of files
    • Allows users to undo a set of changes made by another developer
    • Allows users to merge contributor changes
